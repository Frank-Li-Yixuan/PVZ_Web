# V0.1 Image Generation Prompts

## Original IP constraints

Use only original character, plant, enemy, UI, and effect designs for Sprout & Steel V0.1.

Global negative constraints for every prompt:

```text
Do not copy or imitate any existing copyrighted game character.
Do not reference external game names, character names, studio names, logos, or recognizable styles.
No text, no watermark, no logo.
transparent background, single centered asset.
Readable silhouette at small gameplay size.
Top-down 2.5D game sprite, slight isometric tilt, clean outline, production asset tone.
```
